1The goal & how the chain rule works here

We want ‚ąāL‚ąāőł for every weight őł (W‚āĀ, b‚āĀ, W‚āā, b‚āā). The loss doesn't touch these weights directly, so we chain through every layer between them:

‚ąāL‚ąāőł = ‚ąāL‚ąā(next) ¬∑ ‚ąā(next)‚ąā(next-1) ¬∑ ‚Ä¶ ¬∑ ‚ąā(layer of őł)‚ąāőł
Golden rule: the gradient ‚ąāL‚ąā(thing) has the same shape as that thing ‚ÄĒ so each step is also a matrix reshape. We carry a "running gradient" and multiply it by one new local derivative each step.

Step 1 Loss ‚Üí A‚āā 1√ó1

Forward

d/dA‚āā [ (1‚ąíy)¬∑ln(1‚ąíA‚āā) ] = (1‚ąíy) ¬∑ 11‚ąíA‚āā ¬∑ (‚ąí1) = ‚ąí1‚ąíy1‚ąíA‚āā

‚ąāL‚ąāA‚āā = ‚ąíyA‚āā + 1‚ąíy1‚ąíA‚āā

Step 2 A‚āā ‚Üí Z‚āā  ‚ÄĒ proving the sigmoid derivative 1√ó1

Forward
A‚āā = ŌÉ(Z‚āā) = 11 + e‚ąíZ‚āā = (1 + e‚ąíZ‚āā)‚ąí1
Local derivative ‚ąāA‚āā/‚ąāZ‚āā ‚ÄĒ this is the part you asked about
apply the chain rule to (1 + e‚ąíZ‚āā)‚ąí1  (outer = power ‚ąí1, inner = 1+e‚ąíZ‚āā):
d/du[u‚ąí1] = ‚ąíu‚ąí2, then times the derivative of the inside
‚ąāA‚āā‚ąāZ‚āā = ‚ąí(1 + e‚ąíZ‚āā)‚ąí2 ¬∑ ddZ‚āā(1 + e‚ąíZ‚āā)
ddZ‚āā(1 + e‚ąíZ‚āā) = 0 + e‚ąíZ‚āā¬∑(‚ąí1) = ‚ąíe‚ąíZ‚āā
(derivative of e‚ąíZ‚āā is e‚ąíZ‚āā times the derivative of ‚ąíZ‚āā, which is ‚ąí1)
so   ‚ąāA‚āā‚ąāZ‚āā = ‚ąí(1 + e‚ąíZ‚āā)‚ąí2 ¬∑ (‚ąíe‚ąíZ‚āā) = e‚ąíZ‚āā(1 + e‚ąíZ‚āā)2
now rewrite it using A‚āā itself. Notice:   1 ‚ąí A‚āā = 1 ‚ąí 11+e‚ąíZ‚āā = e‚ąíZ‚āā1+e‚ąíZ‚āā
therefore   A‚āā¬∑(1‚ąíA‚āā) = 11+e‚ąíZ‚āā ¬∑ e‚ąíZ‚āā1+e‚ąíZ‚āā = e‚ąíZ‚āā(1+e‚ąíZ‚āā)2
the two boxed expressions are identical, so   ‚ąāA‚āā‚ąāZ‚āā = A‚āā(1 ‚ąí A‚āā)
Chain rule ‚ÄĒ multiply Step 1 √ó Step 2
‚ąāL‚ąāZ‚āā = ‚ąāL‚ąāA‚āā ¬∑ ‚ąāA‚āā‚ąāZ‚āā = A‚āā‚ąíyA‚āā(1‚ąíA‚āā) ¬∑ A‚āā(1‚ąíA‚āā) = A‚āā ‚ąí y = őī
the A‚āā(1‚ąíA‚āā) on the bottom of Step 1 cancels the A‚āā(1‚ąíA‚āā) from Step 2.
Simple: the sigmoid's slope is A‚āā(1‚ąíA‚āā); when multiplied by the loss derivative it cancels perfectly, leaving the clean error őī = A‚āā ‚ąí y.
