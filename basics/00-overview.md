# CNN Basics — A Learning Path

Welcome! This folder is a beginner-friendly, build-from-intuition course on
**Convolutional Neural Networks (CNNs)**. It explains *what* CNNs are, *why*
they work, *how they mirror the human visual system*, and *what high-level
operations* happen inside them.

Read the files in order. Each one builds on the last, and each ends with a
"key takeaways" box so you can review quickly.

## How to use this material

- **First pass:** Read 1 → 2 → 3 to build intuition. Don't worry about math.
- **Second pass:** Read 4 → 7 to understand the moving parts.
- **Third pass:** Read 8 → 9 to see the whole system and how it learns.
- Keep the [Glossary](10-glossary.md) open in a side tab.

## The one-sentence summary

> A CNN is a layered system that learns to detect simple visual patterns
> (edges, colors) and combine them into complex ones (eyes, wheels, faces),
> roughly the same way your brain's visual cortex processes what your eyes see.

Happy learning! 🧠📸
