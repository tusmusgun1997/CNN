Backpropagation Through Time

A 3-word input has 3 time steps — so the shared weights Wᵢ and Wₕ get three levels of differentiation, one summed path per step. Watch each path flow back.

Vanishing gradient: level 3 (the earliest step) multiplies two f′·Wₕ factors; over T steps it's T−1 of them. Repeatedly multiplying values < 1 shrinks the earliest gradients toward zero — which is exactly why LSTM/GRU exist.
